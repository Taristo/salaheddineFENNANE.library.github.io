مقدمة

C33 تقدم العديد من الخصائص والإمكانيات لإنشاء تأثيرات حركية (أو animations) مذهلة على صفحات الويب. إليك شرح لأهم جوانب تقنية الرسوم المتحركة في CSS3:

1)تكتب كامل animatin داخل keyframes@

■ مثال

@keyframes

animationName

{

0%

{ /* start of animation*/ }

50%

{ /* milio of animation */ }

100%

{ /* end of animation*/}
}

2) يلزمك تسمية العنصر الدي تريد تحريكه واعطائه مدة زمنية يعمل فيها و المدة الزمنية الدي ينتضر قبل عنله و كيف يبدئ و كيف ينتهي و بعدها استدعائه على keyframes@

■ كيف يمكن دالك

او لا يلزمك ان تصنع العصر الدي تريد تحريكه و تضع داخل الستايل الخاص به:

animation-name: animationName;
/*اسم animation*/
animation-duration: 3s;
/*مدة الانميشن*/
animation-timing-function: ease-in-out;
/*طريقة ضهره هل يعني معتدلو او يكون على بطئ حتى يسرع او..*/
animation-delay: 1s;
/*مدة الزمنية قبل بدأ الانميشن*/
animation-iteration-count: infinite;
/*عدد المرات الدي يعاد فيها الانميشن*/
animation-direction: alternate;
/*كيف يعود الانميشن بعد انتهائه*/
animation-play-state: running;
/*حالة الانميشن هل يكون متوقف او يعمل فحالة كنت تريد ان تستدعيه في javascriptاو ان كنت تريد ان تفعل شيئ على hover*/
animation-fill-mode:forwards;
/*حالة animation بعد الانتهاء */

او يمكن تلخيص كل هدا في خانة واحدة

animation: name 1s linear 2 2s reverse running both;

لتحديد المدة الزمنية يمكنك كتابة العدد و ان كان لا نهائية تكتب infinite
لجعل animation متوقف تكتب paused و شغال running
لجعل animationيبقى على حالة 100% عند الانتهاء تكتب forwards
و لتجعله يبقى على حالة البداية backwards
و لتجعله يبدئ كما يكون في البداية و ينتهي كما يكون في النهاية تكتب both
